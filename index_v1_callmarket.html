Bubble Lab

setup

One-time setup for the instructor: connect this page to a free Firebase database so students' phones can sync.

1
Go to console.firebase.google.com → Add project (any name, Analytics off).
2
In the project: Build → Realtime Database → Create database → choose a region → start in test mode.
3
Project settings (gear icon) → Your apps → Web app (</>) → register → copy the firebaseConfig object.
4
Open index.html in any text editor, paste your values into the firebaseConfig block at the top of the script, and re-upload to GitHub.
Make sure the config includes a databaseURL line. If it doesn't appear automatically, copy the URL shown at the top of the Realtime Database page.

          Discussion

          1. Everyone knew the dividend odds, so everyone could compute the fundamental value. Why did the price still leave it?
          2. Look at who finished at the top. Did they trade close to fundamentals, or did they ride the momentum and get out in time?
          3. If we ran this exact market again with the same group, would the bubble be bigger or smaller? (Smith found bubbles shrink with experience.)
          4. What real-world markets does this remind you of, and what was that market's "final period"?
